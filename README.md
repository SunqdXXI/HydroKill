# HydroKill v9.9
Private AI system by SUNQD. Runs 100% locally. No cloud. No tracking.

## Requirements
- Python 3
- Ollama

## Setup

### 1 — Install Ollama
Go to https://ollama.com and install it for your system.

### 2 — Download the brain
ollama pull llama3

### 3 — Clone this repo
https://github.com/SunqdXXI/HydroKill.git


### 4 — Create virtual environment

python -m venv hydrokill-env
source hydrokill-env/bin/activate

### 5 — Install dependencies

pip install -r requirements.txt

### 6 — Run HydroKill

python 1.py

### 7 — Run web version

python HYDROapp.py

Then open http://localhost:5000 in your browser.
